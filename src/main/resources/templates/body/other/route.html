    一、基础知识

    序号 学习内容 课程链接地址 视频地址
    00 Eclipse/IDEA工具的使用 Eclipse/IDEA工具的使用 暂无
    01 J2EE 基础 Java_基础 暂无
    02 Java 23种设计模式 Java_23种设计模式 暂无
    03 MySQL数据库 Mysql_基础 暂无
    04 JDBC JDBC 暂无
    掌握要点

    1. 认识Java、知道如何安装、使用IDE、这里推荐大家使用Eclipse，IDEA也行啊，主要是还得破解。

    2. 学会Java基础知识，知道各种数据类型、集合、线程、IO、泛型、异常等等，都需要熟练掌握，学以致用

    3. 设计模式不着急一下都学会，但是要知道这个设计模式是啥玩意，有多少个，分别是啥，用来干啥的都

    4. 熟练使用数据库，一些常用的命令都得知道啊。还有就是触发器、存储过程、函数，索引，还有视图啥的，都得会

    5. 一定要知道 JDBC 是如何连接各种类型的数据库，如何连接，掌握JDBC调用存储过程、函数，啥的，还有就是要知道事务这个概念(很重要)

    掌握要点

    1. 掌握struts，像它的原理、拦截器、OGNL等等，都要会

    2. 掌握 Hibernate 的工作原理，流程，学会它的映射，以及实体的对应关系，一对多，多对一要能知道。

    4. 一定要掌握springMVC的常用注解，知道这些主键怎么用，区别是啥，以及RestFul风格

    5. 一定要掌握Mybatis的运行机制、配置、动态SQL，尤其是这个动态SQL！！！
